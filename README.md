# Grid

Grid is a Trello clone built with Next.js 16, React 19, TypeScript, Clerk, Supabase, Ant Design and Tailwind CSS. [Inspired by PedroTech tutorial](https://github.com/machadop1407/trello-clone-fullstack).

## Development

Run the development server:

```bash
npm run dev
```

## Tech Stack

- [Next.js 16](https://nextjs.org/)
- [React 19](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Clerk](https://clerk.com/)
- [Supabase](https://supabase.com/)
- [Ant Design](https://ant.design/)
- [Tailwind CSS](https://tailwindcss.com/)
